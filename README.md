# GreenBasket - Premium Specialty Grocery Store Template

A premium, production-ready HTML template for specialty grocery stores. Built with Tailwind CSS, vanilla JavaScript, and designed for ThemeForest and Creative Market.

## Features

- ✅ Fully Responsive (Mobile, Tablet, Desktop)
- ✅ Dark Mode Support with localStorage persistence
- ✅ Clean, Modern Design
- ✅ GSAP Animations
- ✅ Shopping Cart Functionality
- ✅ Theme Toggle
- ✅ Glassmorphism Effects
- ✅ Neomorphism Elements (Micro Use)
- ✅ SEO Optimized
- ✅ Accessible

## Pages Included

### Frontend Pages
- `index.html` - Home v1 (Conversion-focused)
- `home-2.html` - Home v2 (Storytelling/Lifestyle)
- `shop.html` - Product Listing
- `product-details.html` - Product Detail Page
- `categories.html` - Category Listing
- `cart.html` - Shopping Cart
- `about.html` - About Us
- `blog.html` - Blog Listing
- `blog-details.html` - Blog Post Detail
- `contact.html` - Contact Page
- `404.html` - 404 Error Page
- `coming-soon.html` - Coming Soon Page

### Auth Pages
- `auth/login.html` - Login Page
- `auth/signup.html` - Sign Up Page
- `auth/forgot-password.html` - Password Reset

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- GSAP (Animations)
- localStorage (Cart & Theme)

## Setup

1. Open any HTML file in a browser
2. No build process required - uses Tailwind CDN
3. All assets are organized in the `assets/` folder

## Customization

### Colors
Edit CSS variables in `assets/css/style.css`:
- Light mode colors
- Dark mode colors
- Brand colors

### Fonts
- Headings: DM Sans
- Body: Inter
- Both loaded from Google Fonts
